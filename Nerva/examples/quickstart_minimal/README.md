# Quickstart Minimal Example

A 100-line demo showing the basics of Nerva with one model and two tools.

## What This Demonstrates

- Initializing the kernel
- Setting up a model adapter
- Registering tools
- Processing a simple query
- Processing a complex multi-step query

## Running

```bash
cd examples/quickstart_minimal
npm install
npm start
```
