# Agents Tour Example

Demonstrates the planner + executor + summarizer agent loop.

## What This Demonstrates

- Creating a complex goal
- Planner agent breaking it into steps
- Executor agent running the plan
- Summarizer agent compressing results
- Streaming progress updates

## Running

```bash
cd examples/agents_tour
npm install
npm start
```
